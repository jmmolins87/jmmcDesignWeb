Mis Herramientas

Photoshop

Retoque fotográfico, mejora de imágenes, sobre todo retratos de personas para su posterior utilización.
Fotocomposición, montaje de distintas imágenes, con textos y efectos para dar una sensación más impresionante.
Tratamiento de imágenes, cambiar el modo de imágen, curvas, niveles, saturación, etc, para su posterior uso, ya sea impresión o para uso digital como puede ser para una web.

Illustrator

Dibujo vectorial, consiste en crear formas digitales con objetos geométricos dependientes que formarán parte del diseño final del trabajo.
Redibujar, es lo mismo que vectorizar imágenes, con una plantilla bloqueada en el fondo de la mesa de trabajo con la forma que se quiera vectorizaren el momento de crear un diseño.
Vectores en 3D, Consiste crear textos, logos, formas y lo que queramos con relleno que dará la impresión de que salga de nuestro diseño, ya sea en pantalla o en impresión.

Indesign

Diseño gráfico, diseño de carteles, tarjetas, flyer's, dípticos, trípticos, cartas de restaurantes, etc....
Maquetación, edición y diseño de revistas y libros para la posterior impresión o para subirlos a internet.
Pdf interactivo, creación de pdf's interactivos para publicarlos on-line, ya sea en una web o para el envio por correo electrónico.

Dreamweaver

Diseño web, planificación, diseño, implementación y mantenimiento de sitios web, como la navegabilidad, interactividad, usabilidad, arquitectura de la información.
Programación, creación de sitios web para Internet o una intranet. Para conseguirlo se hace uso de tecnologías de software del lado del servidor y del cliente.
Publicación, sistema para poder almacenar información, imágenes, vídeo, o cualquier contenido accesible vía web.

Animate

Dibujo vectorial, se trata de una aplicación de creación y manipulación de gráficos vectoriales
ActionScript, lenguaje de programación, originalmente desarrollado como una forma para que los desarrolladores programen de forma más interactiva
Utiliza, gráficos vectoriales y gráficos rasterizados, sonido, código de programa, flujo de vídeo y audio bidireccional

Html 5

Estructura, es el esqueleto de cualquier página web. Su estructura se organiza con etiquetas, cada una de ellas nos ayuda a organizar el documento.
Sintaxis, no es consideradao un lenguaje de programación por algunos programadores, es un lenguaje simple que estrcutra las páginas web.
Etiquetas, establece una serie de nuevos elementos y atributos que reflejan el uso típico de los sitios web modernos.

Css 3

Estructura, al igual que los documentos HTML, los documentos CSS son archivos de texto donde se escribe una serie de órdenes y el cliente (navegador) las interpreta y aplica a los documentos HTML asociados.
Sintaxis, Selector: El selector es el elemento HTML que vamos a seleccionar del documento para aplicarle un estilo concreto. Propiedad: La propiedad es una de las diferentes características que brinda el lenguaje CSS e iremos aprendiendo.
Valor: Cada propiedad CSS tiene una serie de valores concretos, con los que tendrá uno u otro comportamiento.
Diseño, con css se dá color, forma, organización, estructura e incluso se puede crear keyframes para dar movimiento a algunos elementos que componen el sitio web.

JavaScript

Estructura, la estructura más utilizada en JavaScript y en la mayoría de lenguajes de programación es la estructura if, hay muchas más, también se les llaman bucles.
Sintaxis, no se tienen en cuenta los espacios en blanco y las nuevas líneas, se distinguen las mayúsculas y minúsculas, no se define el tipo de las variables, no es necesario terminar cada sentencia con el carácter de punto y coma (;) y se pueden incluir comentarios.
Dinamismo, el último pilar de la base de toda página web en lo que concierne a su parte visual, estructura y experiencia de usuario.

PHP MySQL

Estructura, al igual que en javaScript la estructura de control más utilizada es el If, pero hay muchas más.
Sintaxis, es un lenguaje que puede estar incrustado en el lenguaje HTML. Por tanto debemos indicar de alguna forma que dejamos de escribir en código HTML y empezamos a escribir en código PHP.
Base de Datos, con php podemos conectar a una base de datos para crear sistemas de registro, crear tiendas on-line donde en la base de datos estarán todos los productos a vender, etc..

Wordpress

Funcionamiento y temas, back-end, barra de herramientas, escritorio, entradas, medios, páginas, comentarios, apariencia, plug-in, usuarios, herramientas. Anatomía de un tema, hojas de estilo, functions.php, archivos de plantilla, temas hijos y temas desundos.
Nucleo interno y técnicas de mejora, Loops y Queries, hooks, base de datos.
Multisite, seguridad, SEO, herramientas, recursos y documentación.
Ampliación personalizada de características, custom Post Type, custom taxonomies, metadatos, shortcodes sidebar widgets y páginas de opciones.

e-Commerce

Plataformas e-Commerce, tipos de plataformas para e-Commerce, comparativa y funcionalidad. WooCommerce y Prestashop.
Logística y legislación, métodos de pago, envíos y devoluciones, atención al cliente y aspectos legales.
Marketing digital, tráfico webm Seo y PPC, redes sociales y e-mail marketing.

Otras pequeñas herramientas que me ayudan en el día a día

Acrobat Reader,
entre sus muchos usos, acrobat es una herramienta muy útil para la pre-impresión, para la gestión de los colores que hay en cada parte del documento entre muchos usos más.

Acrobat Destiller,
pertenece a la familia Adobe Acrobat de productos informáticos. Usado para generar archivos PDF desde archivos PostScript.

NeoStampa,
otro programa para la pre-impresión offset y puerta de enlaze de plotter a ordenador.

Bootstrap,
maquetador html y css con componentes javascript, muy útil para diseños adaptables o responsivos.

jQuery,
es un marco para javascript que nos simplifica extremadamente la programación, tiene una gran variedad de funciones básicas, todas ellas comprimidas en un solo documento.

PHPMailer,
permite mandar correos electrónicos con mucha facilidad y también mandar de manera masiva, com propaganda.

Git Hub,
se utiliza principalmente para la creación de código fuente de programas de computadora. Muy útil para compartir snippets, o partes de código para implementarlo en otro proyecto.

Prestashop,
Creación de una tienda on-line.

Xampp,
Conjunto de programas que permite desenvolupar un sitio web antes de que esté subido al servidor.

Filezilla,
gestor de archivos para subir una web al servidor deseado.

cPanel,
panel de control de cualquier servidor dónde está hospedada una web, sirve para gestionar e-mails, activar certificados ssl, añdir dominios a nuestro servidor, y muchas cosas más

Apache,
es el servidor HTTP, su función es mandar el sitio web, o documentos html, del servidor a su navegador. Se puede intervenir esta transferencia para recabar información.
