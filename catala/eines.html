Les meves Eines

Photoshop

Retoc fotogràfic, millora d'imatges, sobre tot retrats de persones per al seu posterior ús.
Fotocomposició, montatge de diferents imatges, amb texts i efectes per donar una sensació més impressionant.
Tractament d'imatges, canviar la manera d'imatge, corbes, nivells, saturació, etc, per al seu posterior ús, ja sigui impressió o per a ús digital com pot ser per a un web.

Illustrator

Dibuix vectorial, consisteix a crear formes digitals amb objectes geomètrics dependents que formaran part del disseny final del treball.
Redibuixar, és el mateix que vectorizar imatges, amb una plantilla bloquejada al fons de la taula de treball amb la forma que es vulgui vectorizaren el moment de crear un disseny.
Vectors en 3D, consisteix a crear textos, logos, formes i el que vulguem amb farciment que donarà la impressió que surti del nostre disseny, ja sigui en pantalla o en impressió.

Indesign

Disseny gràfic, disseny de cartells, targetes, flyer s, díptics, tríptics, cartes de restaurants, etc ....
Maquetació, edició i disseny de revistes i llibres per a la posterior impressió o per pujar-los a internet.
Pdf interactiu, creació de pdf interactius per publicar-los en línia, ja sigui en un web o per a l'enviament per correu electrònic.

Dreamweaver

Disseny web, planificació, disseny, implementació i manteniment de llocs web, com la navegabilitat, interactivitat, usabilitat, arquitectura de la informació.
Programació, creació de llocs web per a Internet o una intranet. Per aconseguir-ho es fa ús de tecnologies de programari del costat del servidor i del client.
Publicació, sistema per poder emmagatzemar informació, imatges, vídeo, o qualsevol contingut accessible via web.

Animate

Dibuix vectorial, es tracta d'una aplicació de creació i manipulació de gràfics vectorials
ActionScript, llenguatge de programació, originalment desenvolupat com un mètode perquè els desenvolupadors programin de manera més interactiva
Utilitza, gràfics vectorials i gràfics rasterizados, so, codi de programa, flux de vídeo i àudio bidireccional

Html 5

Estructura, és l'esquelet de qualsevol pàgina web. La seva estrucutra s'organitza amb etiquetes, cadascuna d'elles ens ajuda a organitzar el document.
Sintaxi, no és considerat un llenguatge de programació per alguns programadors, és un llenguatge simple que estrcutra les pàgines web.
Etiquetes, estableix una sèrie de nous elements i atributs que reflecteixen l'ús típic dels llocs web moderns.

Css 3

Estructura, de la mateixa manera que els documents HTML, els documents CSS són arxius de text on s'escriu un seguit d'ordres i el client (navegador) les interpreta i aplica als documents HTML associats.
Sintaxi, Selector: El selector és l'element HTML que anem a seleccionar del document per a aplicar-li un estil concret. Propieta: La propietat és una de les diferents característiques que brinda el llenguatge CSS i anirem aprenent.
Valor:Cada propietat CSS té una sèrie de valors concrets, amb els que tindrà un o altre comportament.
Disseny, amb css es dóna color, forma, organització, estructura i fins i tot es pot crear keyframes per donar moviment a alguns elements que componen el lloc web.

JavaScript

Estructura, l'estructura més utilitzada en JavaScript i en la majoria de llenguatges de programació és l'estructura if, hi ha moltes més, també se'ls criden bucles.
Sintaxi, no es tenen en compte els espais en blanc i les noves línies, es distingeixen les majúscules i minúscules, no es defineix el tipus de les variables, no cal acabar cada sentència amb el caràcter de punt i coma (;) i es poden incloure comentaris.
Dinamisme, l'últim pilar de la base de tota pàgina web pel que fa a la seva part visual, estructura i experiència d'usuari.

PHP MySQL

Estructura, igual que en javaScript l'estructura de control més utilitzada és el If, però hi ha moltes més.
Sintaxi, és un llenguatge que pot estar incrustat en el llenguatge HTML. Per tant hem d'indicar d'alguna manera que deixem d'escriure en codi HTML i vam començar a escriure en codi PHP.
Base de Dades, amb php podem connectar a una base de dades per crear sistemes de registre, crear botigues on-line on a la base de dades hi seran tots els productes a vendre, etc ..

Wordpress

Funcionament i temes, back-end, barra d'eines, escriptori, entrades, mitjans, pàgines, comentaris, aparença, plug-in, usuaris, eines. Anatomia d'un tema, fulls d'estil, functions.php, arxius de plantilla, temes fills i temes desundos.
Nucli intern i tècniques de millora, Loops i Queries, hooks, base de dades.
Multisite, seguretat, SEO, eines, recursos i documentació.
Ampliació personalitzada de característiques, custom Post Type, custom Taxonomies, metadades, shortcodes sidebar widgets i pàgines d'opcions.

e-Commerce

Plataformes e-Commerce, tipus de plataformes per e-Commerce, comparativa i funcionalitat. WooCommerce y Prestashop.
Logística i legislació, mètodes de pagament, enviaments i devolucions, atenció al client i aspectes legals.
Marketing digital, trànsit WebM Seu i PPC, xarxes socials i e-mail màrqueting.

Altres petites eines que m'ajuden en el dia a dia

Acrobat Reader,
Entre els seus molts usos, acrobat és una eina molt útil per a la pre-impressió, per a la gestió dels colors que hi ha a cada part del document entre molts usos més.

Acrobat Destiller,
pertany a la família Adobe Acrobat de productes informàtics. Usat per generar arxius PDF des d'arxius PostScript.

NeoStampa,
altre programa per a la pre-impressió offset i porta de enlaze de plotter a ordinador.

Bootstrap,
maquetador html i css amb components javascript, molt útil per a dissenys adaptables o responsius.

jQuery,
és un marc per javascript que ens simplifica extremadament la programació, té una gran varietat de funcions bàsiques, totes elles comprimides en un sol document.

PHPMailer,
permet enviar correus electrònics amb molta facilitat i també enviar de manera massiva, com propaganda.

Git Hub,
és útil sobretot per a la creació de codi font de programes d'ordinador. Molt útil per compartir snippets, o parts de codi per implementar-ho en un altre projecte.

Prestashop,
Creació d'una botiga on-line.

Xampp,
Conjunt de programes que permet desenvolupar un lloc web abans que estigui pujat al servidor.

Filezilla,
gestor de fitxers per pujar un web al servidor desitjat.

cPanel,
panell de control de qualsevol servidor on està allotjada un web, serveix per gestionar correus electrònics, activar certificats SSL, añdir dominis al nostre servidor, i moltes coses més

Apache,
és el servidor HTTP, la seva funció és enviar el lloc web, o documents html, del servidor al seu navegador. Es pot intervenir aquesta transferència per a demanar informació.
